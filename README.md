# PL-1 migration planning session example

## Download
- Download `sample.pli` from: https://github.com/IBM/zopeneditor-sample/blob/main/PLI/PSAM1LIB.pli
- Save it into the workspace root as `sample.pli` (or update paths accordingly).

## Copilot Sessions

### Session 1
#file:sample.pli 
I want to understand what this code does, create a analysis.md file that explains the code structure

----


I want to rearchitect #file:sample.pli file. The analysis of the code is in #file:analysis.md . Give me options and put the options in a reachitect.md file. 

---

 like option 6 in #file:reachitect.md , generate a plan fo rearchitecting file #file:sample.pli into java, call it javamigrationplan.md, include task lists as well so we can track progress

### Session 2 - CTRL+SHIFT+ENTER
I want you to migrate #file:sample.pli  to java using #file:javamigrationplan.md
