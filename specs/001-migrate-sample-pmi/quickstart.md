# Quickstart: PSAM1 Migration CLI

## Prerequisites
- Decide language: Rust or Go preferred; Python possible with hardening
- Install chosen toolchain (Rust: rustup; Go: go toolchain)

## Run (concept)
- Build the CLI
- Execute with transaction and customer file paths
- Outputs: detail lines and totals; logs and metrics emitted

## Security & Observability
- Configure allowlist paths
- Enable structured logging to JSON
- Export metrics to stdout or file
